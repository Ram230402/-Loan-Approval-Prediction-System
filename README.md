# Loan Approval Prediction using Machine Learning

## Project Overview

This project focuses on predicting whether a loan application will be **Approved** or **Rejected** using Machine Learning. The notebook includes complete data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, and prediction.

The goal is to help financial institutions automate the loan approval process by identifying eligible applicants based on their demographic and financial information.
